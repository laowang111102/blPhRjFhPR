# 项目简介 java735

本项目是基于Spring Boot和Vue.js的前后端分离的社团管理系统。以下是对该系统的概览、技术栈、功能模块、系统角色、运行环境、部署步骤以及目录结构的详细介绍。

## 技术栈

- 前端：Vue.js
- 后端：Spring Boot
- 数据库：MySQL

## 功能模块

系统主要分为以下几个功能模块：

1. 用户管理
   - 用户注册、登录、信息修改
2. 社团管理
   - 社团创建、编辑、解散
   - 社团成员管理
3. 活动管理
   - 活动发布、编辑、取消
   - 活动报名、签到
4. 资源共享
   - 文档、图片、视频等资源上传与下载
5. 通知公告
   - 通知发布、查看

## 系统角色

- 系统管理员
- 社团管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：JDK 1.8 或以上版本，MySQL 5.7 或以上版本

## 部署步骤

1. 准备运行环境
   - 安装JDK 1.8 或以上版本
   - 安装MySQL 5.7 或以上版本
   - 创建数据库，导入数据表结构
2. 部署后端
   - 下载源码，使用IDE（如IntelliJ IDEA）打开
   - 配置数据库连接信息
   - 打包项目，运行主程序
3. 部署前端
   - 下载前端源码，解压
   - 在命令行中进入解压后的目录，运行安装依赖命令（如：`npm install`）
   - 运行编译命令（如：`npm run build`）
   - 将编译后的静态文件部署到Web服务器

项目名称/
